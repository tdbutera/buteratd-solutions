Future Project 2

The Suitcase — a church online the day it opens

A pre-built case of network gear, configured before it ships. Plant a church, connect it, and the building has Wi-Fi, a working network, and the bandwidth to livestream — the same day, no IT visit. Same UniFi core inside each; only the way it reaches the internet changes. Every box is owned by the church, with no license to renew.

Future Project 3 · The Vision

Unified fabric — Maui HQ over every site

Maui HQ, run by ButeraNet, sits at the top. Every church — Hawai‘i, the mainland, and the nations — falls beneath it on one simple dashboard. It is a window for visibility and support, not a kill-switch: each site runs on its own, the church owns all the hardware, and nothing stops working if a vendor ever leaves.
