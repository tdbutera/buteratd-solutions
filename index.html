The Model

The Standardized Stack

One build, proven and repeatable

Every ButeraNet site runs the same technology stack, scaled by size. Standardization is what keeps a multi-site ministry manageable, affordable, and reliable.

Network

Full UniFi — gateway, switching, and wireless — with the model scaled to each site. No per-device or per-feature licensing, ever. The church owns the hardware permanently.

Management

One free pane (UniFi Site Manager) for the network across every site; Atera for the endpoints — patching, monitoring, and remote support from a single console.

Server & identity

At campus scale and above: an on-prem server with Active Directory and a NAS for large media, with documents and email in Microsoft 365.

Security baseline

On every site: VLAN segmentation, endpoint protection, MFA and conditional access, backup, and documented credentials — non-negotiable.
